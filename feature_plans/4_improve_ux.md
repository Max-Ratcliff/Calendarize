# Plan for UX Improvements

**Feature:** Improve UX

This is a parent task for several UX improvements. Each will be broken down into its own plan.

*   **Add All events in one click:** When multiple events are detected, provide a button to add all of them to the calendar at once.
*   **Edit button for incorrect detected details:** Allow users to edit the details of a detected event before adding it to their calendar.
*   **Ensure Proper Capitilization:** The title of the event should be properly capitalized.
*   **Improve Invites -> Gcontacts/csv contacts integration?:** Allow users to invite people to the event by connecting to their Google Contacts or uploading a CSV file.
*   **User accounts for historical event storage:** Allow users to create accounts to save and view their past events.
*   **Improve Mobile UI/UX:** The UI should be more responsive and user-friendly on mobile devices.
